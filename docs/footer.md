# Footer
The footer contains kisa information that can change a lot
## Getting Footer Information
- Endpoint : `misc/footer`
- Request : GET
- Response :
```ts
{
  kisa_text : string
  location : string
  phnum_eng : string
  phnum_kor : string
  email : string
  fb_link : string
  insta_link : string
  yt_link : string
}
```
